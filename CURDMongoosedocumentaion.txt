Here is a complete MongoDB / Mongoose Filter Operators Guide with examples. These are used inside find(), findOne(), update(), etc.

1. Comparison Operators
$gt → greater than
Item.find({
  price: { $gt: 100 }
});


SQL equivalent:

WHERE price > 100

$gte → greater than or equal
Item.find({
  price: { $gte: 100 }
});


SQL:

WHERE price >= 100

$lt → less than
Item.find({
  price: { $lt: 500 }
});

$lte → less than or equal
Item.find({
  price: { $lte: 500 }
});

$eq → equal
Item.find({
  price: { $eq: 100 }
});


Same as:

Item.find({ price: 100 });

$ne → not equal
Item.find({
  status: { $ne: "inactive" }
});


SQL:

WHERE status != 'inactive'

2. Range Query (Min and Max)
Item.find({
  price: {
    $gte: 100,
    $lte: 500
  }
});

3. $in → match any value in array
Item.find({
  status: { $in: ["active", "pending"] }
});


SQL:

WHERE status IN ('active', 'pending')

4. $nin → not in array
Item.find({
  status: { $nin: ["inactive", "deleted"] }
});

5. Logical Operators
$and
Item.find({
  $and: [
    { status: "active" },
    { price: { $gt: 100 } }
  ]
});


Same as:

Item.find({
  status: "active",
  price: { $gt: 100 }
});

$or
Item.find({
  $or: [
    { status: "active" },
    { price: { $lt: 50 } }
  ]
});


SQL:

WHERE status='active' OR price<50

$nor
Item.find({
  $nor: [
    { status: "inactive" },
    { price: { $lt: 100 } }
  ]
});

6. Element Operators
$exists

Check if field exists

Item.find({
  quantity: { $exists: true }
});

$type
Item.find({
  price: { $type: "number" }
});

7. Evaluation Operators
$regex → search (LIKE in SQL)
Item.find({
  name: { $regex: "pen", $options: "i" }
});


Matches:

Pen
pen
Pencil


SQL:

WHERE name LIKE '%pen%'

8. Array Operators
$all
Item.find({
  tags: { $all: ["electronics", "mobile"] }
});

$size
Item.find({
  tags: { $size: 2 }
});

9. Not Operator
$not
Item.find({
  price: { $not: { $gt: 100 } }
});

10. Combining Multiple Filters
Item.find({
  status: "active",
  price: {
    $gte: 100,
    $lte: 1000
  },
  category: {
    $in: ["electronics", "books"]
  },
  name: {
    $regex: "phone",
    $options: "i"
  }
});

8. Projection (Field Selection)

Controls which fields to return.

Include fields

  "name": "Pen",

Exclude fields

9. Sorting Documentation

Sort defines order of results.

Syntax
Model.find().sort(sortObject)

Ascending
Item.find().sort({ price: 1 });

Descending
Item.find().sort({ price: -1 });

Multiple Sort
Item.find().sort({
  price: 1,
  quantity: -1
});

String Format
Item.find().sort("price -quantity");

10. Limit Documentation

Limits number of results.

Syntax
Model.find().limit(number)

Example
Item.find().limit(5);

11. Skip Documentation

Used for pagination.

Syntax
Model.find().skip(number)

Example
Item.find().skip(10);


Skips first 10 records.

12. Pagination Documentation
Formula
skip = (page - 1) * limit

Example
const page = 2;
const limit = 5;

Item.find()
.skip((page - 1) * limit)
.limit(limit);

13. Count Documentation

Returns total count.

Syntax
Model.countDocuments(filter)

Example
Item.countDocuments({
  status: "active"
});

14. Options Documentation
lean()

Returns plain JS object instead of mongoose document.

Item.find().lean();


Benefits:
• Faster
• Less memory

exec()

Executes query explicitly.

Item.find().exec();

new option (update)
Item.findByIdAndUpdate(id, update, {
  new: true
});


Returns updated document.

runValidators
Item.findByIdAndUpdate(id, update, {
  runValidators: true
});

15. Complete Production Example
const getItems = async (req, res) => {

  const {
    page = 1,
    limit = 10,
    sort = "-createdAt",
    status,
    category,
    minPrice,
    maxPrice,
    search,
    select
  } = req.query;


  const filter = {};

  if (status)
    filter.status = status;

  if (category)
    filter.category = category;

  if (minPrice || maxPrice) {
    filter.price = {};

    if (minPrice)
      filter.price.$gte = Number(minPrice);

    if (maxPrice)
      filter.price.$lte = Number(maxPrice);
  }

  if (search) {
    filter.name = {
      $regex: search,
      $options: "i"
    };
  }


  const skip = (page - 1) * limit;


  let query = Item.find(filter);

  if (select)
    query = query.select(select);

  query = query
    .sort(sort)
    .skip(skip)
    .limit(limit)
    .lean();


  const data = await query;

  const total = await Item.countDocuments(filter);


  res.json({
    total,
    page: Number(page),
    limit: Number(limit),
    results: data.length,
    data
  });

};
